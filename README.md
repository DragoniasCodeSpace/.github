# 🐉 Dragonia

Welcome to **Dragonia**, an organization focused on creating fun, interactive **Twitch extensions** that enhance viewer engagement and bring communities closer together.

---

## 🎯 Our Mission

At Dragonia, we aim to transform passive viewing into interactive experiences. By combining creativity with technology, we build extensions that allow streamers and their communities to actively participate in the content.

---

## 🚀 Current Project: Egg Gacha

Our first project is currently in development:

### 🥚 Egg Gacha
- A simple, interactive gacha-style system for Twitch  
- Viewers can participate directly through the stream  
- Designed to be fun, rewarding, and easy to use  

This is just the beginning. The current version focuses on core mechanics, with plans to expand functionality and depth over time.

---

## 🔮 Future Plans

We have several ideas lined up to grow Dragonia into something bigger:

- More advanced gacha systems (rarities, collections, upgrades)  
- Viewer-driven mini-games and events  
- Integration with streamer rewards and loyalty systems  
- Expanded customization for streamers  

---

## 🛠️ Tech Focus

While specifics may evolve, our work generally involves:

- **Frontend:** Interactive UI for Twitch extensions  
- **Backend:** APIs and data handling  
- **Real-time:** Interaction between viewers and streams
